# 泽远跨境 POA 智能申诉系统 V7

## V7 新增功能

### 优先级一：直接提升通过率

**1. 两阶段 POA 生成（新核心流程）**
- 第一步：AI 先生成大纲（JSON 结构化），包含各章节标题和关键要点
- 员工可在大纲阶段直接编辑每个要点，调整论证方向
- 第二步：确认大纲后，AI 严格按照批准的大纲展开完整 POA
- 优势：避免整篇跑偏再重生成；员工介入节点更早；策略控制更精准

**2. 成功案例自动入库**
- 当案件状态标记为「申诉成功」时，系统自动弹出保存弹窗
- 预填标题，一键保存至成功案例库
- 自动打「自动入库」标签并记录来源 Case ID

### 优先级二：提升团队协作效率

**3. 店铺档案记忆**
- 员工填完公司名/店铺名/供应链后，可一键保存为店铺档案
- 下次处理同一客户案件，点击档案标签即可快速填入所有信息
- 支持多店铺管理，可删除不需要的档案

**4. 管理员批注系统**
- 审核案件时，管理员可添加内部批注（员工可见，Walmart 看不到）
- 批注可标记「需要跟进」，提醒员工及时处理
- 员工处理后可标记「已完成」，形成完整的沟通闭环
- 待跟进批注数量会在仪表盘和历史列表中突出显示

### 优先级三：智能学习闭环

**5. 失败案例 AI 分析**
- 申诉失败的案件可一键触发 AI 分析
- AI 从根本原因、关键缺失点、改进建议、成功案例特征四个维度给出中文分析报告
- 分析结果永久保存在案件详情中，审核时可随时查看

**6. 策略自动迭代**
- 设置页面新增「AI 自动迭代策略」按钮
- 基于所有已标记「申诉成功」的案件，AI 分析共同特征
- 分别优化 Performance/ODR、IP/Counterfeit、通用三套策略
- 展示新旧策略对比，管理员可选择应用或放弃

## 启动

```bash
npm install
npm run dev
```
